Volantis

A Wonderful Theme for Hexo 4.2+

Volantis（原名 Material X），一个高度模块化和可定制化、功能相对完善的Hexo博客主题，既可以用作个人博客，也适合用来搭建文档。

这些是使用了本主题的博客： Issue #97

环境要求

如果您从旧版本更新或着其它主题迁移，请确保环境版本不要太低，否则会产生兼容性问题。

环境 建议最低版本
Hexo 4.2
hexo-cli 3.1
node.js 12.16
npm 6.13
